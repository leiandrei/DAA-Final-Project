# DAA-Final-Project
This is a Final Project for Design and Analysis of Algorithms. It is a web-based application or dashbaord that visualizes single source shortes path search using algorithmic techniques like Dijkstra's Algorithm using Geospatial features. 

## Project Structure
```
├── notebooks/          # Jupyter notebooks for algorithm prototyping and EDA
├── src/                # Core application source code
│   ├── algorithms/     # Dijkstra's implementation and helper logic
│   ├── utils/          # Geospatial data processing and map rendering
│   └── components/     # UI elements for the web dashboard
├── app.py              # Main entry point (Streamlit/Flask/FastAPI)
├── requirements.txt    # Project dependencies
├── .gitignore          # Files to exclude from version control
└── README.md           # Project documentation
```

## Prerequisites
``
Python 3.8+
``

## Install Dependencies
```
pip install -r requirements.txt
```

## Clone the Repository
```
git clone https://github.com/your-username/DAA-Final-Project.git
cd DAA-Final-Project
```

## Run the Application
```
streamlit run app.py
```
